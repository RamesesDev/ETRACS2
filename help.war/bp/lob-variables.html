Managing LOB Variable

This facility enables you to add, edit, and delete a LOB Variable.

How to add a LOB Variable?

1. On the menu toolbar, go to "Business Permit -> Master -> Variable" item.



2. Click on the "New" button.



3. And fill in the Name, Caption, Description, and Data type fields.
4. Then Click on the "Save" button.





How to edit a LOB Variable?

1. Go to "Line of Business" page. (Please refer to the Added LOB Variable - Step 1)
2. Select an item from the lob variable list, and click on the "Edit" button.
3. Modify the selected LOB Variable item.



4. Then click "Save" button.




How to delete a LOB Variable?

1. Go to "Line of Business" page. (Please refer to the Added LOB Variable - Step 1)
2. Select an item from the lob variable list, and click on the "Delete" button.
